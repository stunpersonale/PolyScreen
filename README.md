# PolyScreen
Python script used to get color values of each pixel in a low-quality screenshot of your screen. Used for Polytoria Screensharing.

### Installation
- **Download the .zip and unzip it anywhere.**
- **Get Python from https://www.python.org/downloads/. Any new version should work fine, but 3.11.6 is the version it was written for.**
- **Go to the folder generated by the zip file in file explorer and write cmd in the top bar.**
