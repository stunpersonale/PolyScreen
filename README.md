# PolyScreen
Python script used to get color values of each pixel in a low-quality screenshot of your screen. Used for Polytoria Screensharing.

### Guide
- **Download the .zip and unzip it anywhere.**
- **Get Python from https://www.python.org/downloads/. Any new version should work fine, but 3.11.6 is the version it was written for.**
- **Go to the folder generated by the zip file in file explorer and write cmd in the top bar.**
   ![Where you're supposed to look, bud.](/readmeimages/1.png)
- **Run this command**
  `pip install -r requirements.txt`
- **Keep the tab open, go to your browser**
- **Go to https://dashboard.ngrok.com/signup**
- **Make an account, go to the dashboard**
- **Go to the sidebar, click on "Your Authtoken"**
- **Copy it.**
  ![Where you're supposed to look, bud.](/readmeimages/2.png)
- **Go back to the cmd prompt and write this.**
  `python main.py` (if it doesnt work try `py main.py` or `python3 main.py` or any of those with the -m flag (for example `py -m main.py`)
- **Put in the Authtoken from ngrok**
- **If you did it correctly it should print out a link that looks something like https://chicken-uncle-doorstopper.ngrok-free.dev/api/pixels**
- **Copy it and paste it into the Polytoria game input in the Endpoint URL.**
